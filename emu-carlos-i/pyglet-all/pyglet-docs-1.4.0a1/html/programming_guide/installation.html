Installation¶

Note

These instructions apply to pyglet 1.4.

To play compressed audio and video files (anything but WAV), you will need FFmpeg.

pyglet is available on PyPI. You can install it like any other Python library:

pip install pyglet

If you’re installing pyglet into your global site-packages directory, you might need to add sudo. But you shouldn’t do that; instead, create a venv. Alternatively, you can simply copy pyglet directly into your project folder, and use it without any installation.

Running the examples¶

The source code archives include examples. Archives are available on BitBucket:

wget https://bitbucket.org/pyglet/pyglet/downloads/pyglet-1.3.0.zip
tar -xzvf pyglet-1.3.0.zip
cd pyglet-1.3.0
python examples/graphics.py
